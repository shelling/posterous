# Posterous

Posterous.com API in Perl.


## INSTALLATION

To install this module type the following:

    perl Makefile.PL
    make
    make test
    make install

## DEPENDENCIES

    LWP::UserAgent
    HTTP::Request
    MIME::Base64
    Data::Dumper
    Rubyish::Attribute
    XML::Simple
